# Simple AI Chatbot

This project is a simple AI chatbot built using **Python**, **TensorFlow**, and **NLTK**. The chatbot is designed to understand basic user intents and provide pre-defined responses.

---

## **Project Structure**

* **`intents.json`**: This file contains the chatbot's knowledge base. It's a JSON file where each **intent** is defined by a unique **tag**, example **patterns** (user phrases), and a list of possible **responses**.
* **`train_chatbot.py`**: This is the **training script**. It processes the `intents.json` file, cleans the data, creates a numerical representation (a "bag of words"), and trains a neural network model. The trained model and pre-processed data are saved to disk.
* **`chatbot.py`**: This is the **runtime script**. It loads the trained model and data files. It provides a command-line interface where you can type messages, and the chatbot will predict your intent and provide a response.
* **`words.pkl`**: A file generated by `train_chatbot.py` that stores the chatbot's **vocabulary**.
* **`classes.pkl`**: A file generated by `train_chatbot.py` that stores the list of unique **intent tags**.
* **`chatbot_inimodel.h5`**: The trained **neural network model** saved in H5 format.

## **How to Run the Chatbot**

1.  **Set up the environment:**
    * Make sure you have **Python 3.9** or a compatible version installed.
    * It's recommended to use a virtual environment. From your terminal, run:

    ```bash
    python3 -m venv chatbot
    ```

    * Activate the virtual environment:

    ```bash
    source chatbot/bin/activate
    ```

2.  **Install the required libraries:**
    * With the virtual environment active, run the following command to install all necessary packages:

    ```bash
    pip install tensorflow numpy nltk
    ```

3.  **Download NLTK data:**
    * Open a Python terminal within your virtual environment and download the required NLTK data packages:

    ```python
    import nltk
    nltk.download("punkt")
    nltk.download("wordnet")
    nltk.download("omw-1.4")
    ```

4.  **Train the model:**
    * From your terminal, run the training script. This may take a few minutes.

    ```bash
    python new.py
    ```

    * Upon successful completion, you will see a message: "**Training complete and model saved to chatbot_ini_model.h5**".

5.  **Run the chatbot:**
    * Start the chatbot by running the main script:

    ```bash
    python chatbot.py
    ```

    * You will see the message "**Great!, Bot is running!**" You can now start typing messages and interacting with your bot.

## **Adding More Intents**

You can easily expand the chatbot's capabilities by adding new intents to the **`intents.json`** file.

1.  Open **`intents.json`**.
2.  Add a new JSON object to the **`intents`** array, following the same structure.
3.  Include a unique **`tag`**, several **`patterns`**, and a list of **`responses`**.
4.  After saving the file, you must **retrain the model** by running `python train_chatbot.py` again. This will update the model's knowledge to include the new intent.
